Đây là 1 projects nhỏ để hiểu hơn về các react-hooks.

Cài đặt ứng dụng:
```
 1. git clone https://github.com/tcgnam/react-hooks-simple.git
 2. cd react-hooks-simple
 3. npm install
 4. npm run dev
```
